# zephyr-nvim
A dark neovim colorscheme written in lua and syntax based on
[nvim-treesitter](https://github.com/nvim-treesitter/nvim-treesitter)

### Usage

```lua
lua require('zephyr')
```
or
```vim
colorscheme zephyr
```
